# Build requirement
- C++11
- experimental/filesystem support
- pragma support (pragma once)


# Build / Run on Linux (CMake)
- [CMake](https://cmake.org/)
- Use flags c++11 and lstdc++fs (for experimental/filesystem)

```
mkdir build
cd build
cmake ..
make -j4
make run
```
